# Bookmarks Data Module for Soukai

## Development
```bash
git clone https://github.com/solid-contrib/data-modules
cd data-modules
cd bookmarks/inrupt
npm install
npm run test
```

## Demo app
Assuming you just ran the 'development' steps above and are now in the bookmarks/inrupt folder.
```bash
npm install
npm run build
cd demo
npm install
npm run dev
```
